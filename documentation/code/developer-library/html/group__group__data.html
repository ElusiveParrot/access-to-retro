typedef uintmax_t ar_max_int

Type representing widest possible integer for this platform.

This is useful for functions that perform bitwise operations and allow integer of any size to be changed, this type helps with that as any integer type can be promoted to this one.

Remarks
This type is unsigned, for signed equivalent see 'ar_max_sint'

◆ ar_max_sint

typedef intmax_t ar_max_sint

Type representing widest possible integer for this platform.

This is useful for functions that perform bitwise operations and allow integer of any size to be changed, this type helps with that as any signed integer type can be promoted to this one.

Warning
'ar_max_int' is generally preferred and should always be used if promotion from integers larger than 32-bit is possible
Remarks
This type is signed, for unsigned equivalent see 'ar_max_int'
