Define the virtual console for Access to Retro library.

Should always be called in '_ar_vc_define' function that needs to be defined by the virtual console

Warning
If NULL is provided as any of the string arguments then a default value of UNKNOWN will be used unless it's the 'rom_ext' in which case '*.rom' will be used. It is not advisable to use NULL for any of the values.
Parameters
nameName of the virtual console
systemSystem that is emulated
authorAuthor of the virtual console
rom_extExtension of the ROM files, for example: '*.nes'
versionCurrent version of the virtual console
Returns
Error codes:
  • 0: No error
