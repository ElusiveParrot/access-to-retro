Enumeration Type Documentation

◆ ar_graphics_method

enum ar_graphics_method

Contains all methods of supported graphical output, should be set using 'ar_graphics_set_method'.

Enumerator
ar_graphics_method_none 

No graphical output.

ar_graphics_method_frame_buffer 

Output graphics using a raw frame buffer.

ar_graphics_method_sdl 

Output graphics using SDL2 renderer.

ar_graphics_method_open_gl_context 

Output graphics using OpenGL directly.
