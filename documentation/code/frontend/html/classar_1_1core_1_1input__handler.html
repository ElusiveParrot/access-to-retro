Map used for input translation Qt -> Unified Controller.

Whenever a button is pressed it will be translated from its string value into the enum of unified controller:

KEY_NAME (is =) CURRENTLY_PRESSED_KEY MAP[KEY_NAME] (gives ->) CONTROLLER_KEY CONTROLLER_KEY (is) MARKED AS PRESSED
