AR_DEFINE_REQUIRED_FN ( AR_THREAD_INPUT_FN  )

Input thread function.

Runs when the virtual console is started, should contain an infinite loop (it will stop automatically when the user closes the window), while the developer is free to use this thread as he likes (even to create more threads) it is recommended to use this thread for input handling

Warning
SDL should NOT be used to handle input by the developer, only access to retro library helpers should be used this is due to the fact that SDL does not allow to poll events outside of the thread where it was initialised and doing so will crash the application.
Remarks
This function will loop in a loop in a thread every frame time (1000 / frame rate) (for example: every 16.6ms if the frame rate is 60)
This function NEEDS to be defined for the Access to Retro library.

AR_DEFINE_REQUIRED_FN ( AR_THREAD_MAIN_FN  )

Main thread function.

Runs when the virtual console is started, should contain an infinite loop (it will stop automatically when the user closes the window), while the developer is free to use this thread as he likes (even to create more threads) it is recommended to use this thread for main fetch execute decode cycle and inter-component communication (for ex: CPU <-> GPU).

Remarks
This function will loop in a loop in a thread every frame time (1000 / frame rate) (for example: every 16.6ms if the frame rate is 60)
This function NEEDS to be defined for the Access to Retro library.

AR_DEFINE_REQUIRED_FN ( AR_THREAD_RENDER_FN  )

Render thread function.

Runs when the virtual console is started, should contain an infinite loop (it will stop automatically when the user closes the window), while the developer is free to use this thread as he likes (even to create more threads) it is recommended to use this thread for rendering, whether that's OpenGL, SDL2 or direct frame buffer access.

Remarks
This function will loop in a loop in a thread every frame time (1000 / frame rate) (for example: every 16.6ms if the frame rate is 60)
This function NEEDS to be defined for the Access to Retro library.
