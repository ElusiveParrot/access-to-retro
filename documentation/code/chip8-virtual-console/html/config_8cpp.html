AR_DEFINE_REQUIRED_FN ( AR_DEFINE_FN  )

Defines virtual console's metadata for the frontend to use.

Runs when the virtual console is loaded by the frontend (not when it's started).

Remarks
Graphics mode should also be configured here, using 'ar_graphics_set_method'
This function NEEDS to be defined for the Access to Retro library.
Returns
Error code:
  • 0: No error,
  • Anything else: In this virtual console it's result from ar_define.

AR_DEFINE_REQUIRED_FN ( AR_QUIT_FN  )

Clean up resources of the virtual console and library.

AR_DEFINE_REQUIRED_FN ( AR_STARTUP_FN  )

Configures the virtual console and the developer library.

Runs when the virtual console is started, should be used to configure virtual console's internal state and the library.

Remarks
This function NEEDS to be defined for the Access to Retro library.

  • 0: No error
